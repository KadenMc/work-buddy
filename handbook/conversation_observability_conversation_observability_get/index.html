    Conversation Observability Get¶

    Look up a single observed-session row by session_id, including its metadata (start/end, message_count, span count, tool usage counts). Returns None when the session has not been observed yet.

    MCP name: conversation_observability_get

    Category: conversation_observability

    Parameters¶

    Name Type Required Description
    session_id str Yes Full or 8-char prefix session UUID.
